# Creación de feed RSS y Atom para blog de viajes 

En este proyecto se crearon dos feeds, RSS y Atom, para el blog **"Explora Ciudades"**, un sitio con guías, tips y experiencias para recorrer ciudades de manera práctica y divertida.

Se agregaron cuatro artículos con destinos populares y rutas gastronómicas, siguiendo los estándares de **RSS** y **Atom**. 


## Archivos incluidos

- `viajes.rss`: en formato RSS.
- `viajes.atom`: en formato Atom.  


## Validaciones W3C

Ambos archivos fueron revisados usando: https://validator.w3.org/feed/
